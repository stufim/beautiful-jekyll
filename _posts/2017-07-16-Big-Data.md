We're distinguishing two types of node in Apache Hadoop Cluster
* Master Node (MN)
  - Name Node (NN)
  - Secondary Name Node (SNN)
  - JobTracker (JT)
* Slave Node (SN)
  - Data Node (DN)
  - Task Tracker (TT)
  
  Master node is Single Point Of Failure (SPOF) and Slave nodes depends on Master Node.
